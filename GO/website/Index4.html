Forces:
Gravitation: Ft = m · g ⇔ m = Ft / g

Chemistry:

Mængdeberegning:
n = m/M ⇔ M = m / n ⇔ m = n · M
concentration:
c = n / V ⇔ n = c · V ⇔ V = n / c
Idealgasligningen:
p · V = n · R · T ⇔ n = (p · V) / (R · T) ⇔ V = (n · R · T) / p
