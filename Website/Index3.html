Welcome to project RAM

What is project RAM?

The game

Project RAM is my first atempt at making an online game with C# networking in unity.
The point of the game is simply that you as the player are a cube, Who has to collide with the other players.

How it works

The gameplay is quite simple, all players move forward automaticaly and can choose to either steer using arrow keys, A and D or a joystick of some kind
